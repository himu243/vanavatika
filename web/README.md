# Vanavatika — web (React + Vite)

This folder contains a minimal React app scaffolded for the existing static site.

Quick start:

```bash
cd web
npm install
npm run dev
```

Build:

```bash
npm run build
npm run preview
```

Notes:
- Favicons and `site.webmanifest` are kept in the `favicons/` folder.
- Styles were migrated into `src/index.css` and are imported from `src/main.jsx`.
